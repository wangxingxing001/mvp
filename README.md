### 属性介绍

anim_direction : 是选择顺时针还是逆时针 有俩个参数 positive  and  negative<br/>
anim_duration  : 是动画的时间 默认是200 需要在xml文件中自己配置<br/>
bg_color           : 背景的颜色填充默认是白色<br/>
btn_color          : 按钮图标的颜色<br/>
gap_width          : 内部行条的宽度<br/>
